# Time Complexity

1. Array (unsorted)
    - Search     O(n)
    - Insert     O(1)
    - Delete     O(n)
    - FindClosest O(n)
    - SortedTraversal O(nlogn)

2. Array (sorted)
    - Search     O(log n)
    - Insert     O(n)
    - Delete     O(n)
    - FindClosest O(log n)
    - SortedTraversal O(n)

 3. Linked List
    - Search     O(n)
    - Insert     O(1)
    - Delete     O(n)
    - FindClosest O(n)
    - SortedTraversal O(n log n)

4. BST(balanced)
    - Search     O(log n)
    - Insert     O(log n)
    - Delete     O(log n)
    - FindClosest O(log n)
    - SortedTraversal O(n)

5. HashTable
    - Search     O(1)
    - Insert     O(1)
    - Delete     O(1)
    - FindClosest O( n)
    - SortedTraversal O(n log n)
